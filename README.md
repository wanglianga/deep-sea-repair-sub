# 深海维修潜航闯关游戏

Deep Sea Repair Submarine - 基于 Phaser 引擎的深海维修潜航闯关游戏

## 原始需求

> 请用 Phaser 制作深海维修潜航闯关游戏，玩家驾驶小型潜航器在海底管道、信号塔、破损舱段和母船之间执行维修任务。潜航器拥有氧气、电量、机械臂耐久、照明范围和压力承受值，暗流会推偏移动轨迹，碎石、漏电区和低能见度会影响判断。玩家需要在压力计升高前完成清障、焊接、接线、取样和返航，界面必须让氧气、任务点、危险区、机械臂状态和回船方向一眼可读。

## 项目简介

玩家驾驶小型潜航器深入海底，执行一系列维修任务。游戏包含丰富的环境要素和资源管理机制，玩家需要在氧气、电量、压力等多重限制下完成清障、焊接、接线、取样四项维修任务，然后安全返回母船。

## 技术栈

- **游戏引擎**: Phaser 3
- **构建工具**: Vite 5
- **开发语言**: JavaScript (ES6+)

## 游戏特性

### 潜航器系统
- **氧气系统**: 随时间消耗，耗尽则任务失败
- **电力系统**: 移动、照明、机械臂都会消耗电量
- **压力系统**: 深度越深压力增长越快，达到极限则船体破裂
- **机械臂**: 用于执行维修任务，有耐久度限制
- **照明系统**: 可开关的照明灯，影响视野范围

### 环境系统
- **暗流**: 会推动潜航器偏离航线
- **碎石区**: 碰撞会造成机械臂和氧气损伤
- **漏电区**: 进入会持续损失电力
- **低能见度区**: 视野严重受限

### 任务系统
1. **清障任务** ⛏ - 清理管道中的障碍物
2. **焊接任务** 🔧 - 焊接破损的管道接口
3. **接线任务** ⚡ - 修复信号塔的电路
4. **取样任务** 🧪 - 从破损舱段采集样本
5. **返航任务** 🏠 - 完成所有任务后返回母船

### HUD 界面
- 左侧面板：氧气、电量、压力、机械臂状态
- 右侧面板：任务进度列表
- 右下角：回船方向指南针
- 危险警告：进入危险区域时全屏警示

## 启动方式

### 前置要求

- Node.js >= 16.0.0
- npm 或 pnpm

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

访问地址：http://localhost:3000

#### 3. 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

#### 4. 预览生产构建

```bash
npm run preview
```

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |

## Docker 一键启动

### 前置要求

- Docker

```bash

```

```bash

```

## 游戏攻略提示

1. **注意氧气消耗**：不要在深海停留太久，及时规划路线
2. **避开危险区域**：漏电区和碎石区会造成额外损伤
3. **合理使用照明**：照明灯消耗电力，在安全区域可以关闭
4. **机械臂耐久**：执行任务前确保机械臂有足够耐久
5. **注意压力值**：深海区域压力增长快，不要长时间停留
6. **利用暗流**：顺流而行可以节省电力

## 注意事项

- 游戏使用 Canvas 渲染，建议在现代浏览器中运行
- 最佳体验请使用 1280x720 以上分辨率的屏幕
- 所有游戏资源均为程序生成，无需额外资源文件
