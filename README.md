# 深海维修潜航闯关游戏

Deep Sea Repair Submarine - 基于 Phaser 引擎的深海维修潜航闯关游戏

## 原始需求

> 请用 Phaser 制作深海维修潜航闯关游戏，玩家驾驶小型潜航器在海底管道、信号塔、破损舱段和母船之间执行维修任务。潜航器拥有氧气、电量、机械臂耐久、照明范围和压力承受值，暗流会推偏移动轨迹，碎石、漏电区和低能见度会影响判断。玩家需要在压力计升高前完成清障、焊接、接线、取样和返航，界面必须让氧气、任务点、危险区、机械臂状态和回船方向一眼可读。

## 扩展需求

> 扩展暗流与锚点操作：某些海沟水流会把潜航器推离目标，玩家可以伸出机械臂抓住锚点稳定机身，但会消耗耐久并增加返航风险；熟练玩家能利用暗流节省电量。
> 扩展多步骤维修关卡：管道裂缝需要先清理碎屑、再对准焊点、最后检测压力，信号塔则要重接线路并校准频率，漏掉步骤会在返航时触发二次故障或减少奖励。

## 项目简介

玩家驾驶小型潜航器深入海底，执行一系列维修任务。游戏包含丰富的环境要素和资源管理机制，玩家需要在氧气、电量、压力等多重限制下完成管道维修、信号塔维修等多步骤任务，利用暗流节省电量，使用锚点稳定机身，最终安全返回母船。

## 游戏特性

### 潜航器系统
- **氧气系统**: 随时间消耗，耗尽则任务失败
- **电力系统**: 移动、照明、机械臂都会消耗电量
- **压力系统**: 深度越深压力增长越快，达到极限则船体破裂
- **机械臂**: 用于执行维修任务和抓取锚点，有耐久度限制
- **照明系统**: 可开关的照明灯，影响视野范围

### 环境系统
- **普通暗流**: 蓝色水流，会轻微推动潜航器偏离航线
- **海沟暗流**: 紫色强水流，推力是普通暗流的2.5倍
- **顺流省电**: 顺着暗流方向移动可节省最高70%电量
- **碎石区**: 碰撞会造成机械臂和氧气损伤
- **漏电区**: 进入会持续损失电力
- **低能见度区**: 视野严重受限

### 锚点系统
- **锚点抓取**: 按E键抓取附近的锚点以稳定机身
- **稳定效果**: 锚定后可抵消85%的暗流推力
- **耐久消耗**: 锚定状态会持续消耗机械臂耐久
- **返航风险**: 每次使用锚点会增加返航故障风险

### 多步骤任务系统
1. **管道裂缝维修** 🔧 - 三步骤任务
   - 步骤1：清理碎屑 🧹
   - 步骤2：对准焊点 🎯
   - 步骤3：压力检测 📊
2. **信号塔维修** 📡 - 两步骤任务
   - 步骤1：重接线路 🔌
   - 步骤2：校准频率 📡
3. **清障任务** ⛏ - 单步骤任务
4. **取样任务** 🧪 - 单步骤任务
5. **返航任务** 🏠 - 完成所有任务后返回母船

### 质量与惩罚系统
- **任务质量**: 完整执行所有步骤可获得100%质量
- **跳过步骤**: 跳过步骤会降低质量并增加二次故障风险
- **二次故障**: 返航时可能触发二次故障，消耗额外氧气和电力
- **综合评分**: 根据任务质量、剩余资源、风险值综合评定S/A/B/C/D/F等级

### HUD 界面
- 左侧面板：氧气、电量、压力、机械臂状态
- 右侧面板：任务进度列表
- 右下角：回船方向指南针
- 底部状态栏：锚点状态、省电百分比、风险值
- 危险警告：进入危险区域时全屏警示

## 启动方式

### 前置要求

- Node.js >= 16.0.0
- npm 或 pnpm

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

访问地址：http://localhost:3000

#### 3. 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

#### 4. 预览生产构建

```bash
npm run preview
```

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格键 | 使用机械臂 / 执行任务（按住） |
| E | 抓取/释放锚点 |
| L | 开关照明灯 |

## 游戏攻略提示

1. **注意氧气消耗**：不要在深海停留太久，及时规划路线
2. **避开危险区域**：漏电区和碎石区会造成额外损伤
3. **合理使用照明**：照明灯消耗电力，在安全区域可以关闭
4. **机械臂耐久**：执行任务前确保机械臂有足够耐久
5. **注意压力值**：深海区域压力增长快，不要长时间停留
6. **利用暗流省电**：顺流而行可以节省电力，最高节省70%
7. **海沟区域用锚点**：在强海沟暗流区域附近使用锚点稳定机身
8. **完成所有步骤**：多步骤任务不要跳过步骤，否则会降低质量并增加返航风险
9. **平衡风险与收益**：频繁使用锚点会增加返航风险，需合理规划
